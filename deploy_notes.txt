Carpeta: Deployment

1. Copiar el archivo package.json
2. Instalar las librerías (yarn install)
3. Copiar el resto de archivos del código fuente
4. Compilar el código fuente (yarn build)

Carpeta: Production

5. Copiar la carpeta node_modules desde la carpeta Deployment
6. Copiar la carpeta dist
7. Copiar el archivo package.json desde la carpeta Deployment
8. Ejecutar el script prod (yarn prod)
